# RepoMaster

A structured workspace builder and protocol generator for developers, designed to enforce clean documentation and atomic git commits.

## Tech Stack

- **Framework:** Angular 21 (Standalone, Strict-Mode)
- **Styling:** Tailwind CSS v4
- **State Management:** NgRx SignalStore
- **Database & Auth:** Supabase (PostgreSQL)

## Local Setup

1. Clone the repository
2. Run `npm install`
3. Add your Supabase credentials to `environment.ts` (Setup guide follows)
4. Run `npm start` to boot the development server on `http://localhost:4200/`
